Troubleshooting

If you don't have the latest DOF, or you're having any problems with Visual Pinball launching tables, click here. To take full advantage of the V2 Pinscape firmware, you need the latest version of DOF R3. Here's a link to the latest official build: Build 5845.
You might also need a patch. If you have any problems with VP getting stuck when launching tables, download my unofficial patched version of DirectOutput.dll: DirectOutputPatch.zip. Install the official DOF R3 first, then replace DirectOutput.dll with the one in the patch.
