DOF R3 Update

Latest Updates
If you're using DirectOutput Framework, you should upgrade to the latest version at the link above.
You should also update your LEDWIZ.DLL file if you have that installed. This is used by older, non-DOF software (such as LEDBlinky and Future Pinball) to access LedWiz features. The "stock" LEDWIZ.DLL can make these programs crash if you have any of the Pinscape V2 keyboard features enabled.
You should update these files even if you're not using the Pinscape output controller features. The older versions can crash when merely scanning for devices if they run into a Pinscape unit with the keyboard enabled, so they can crash even if they won't end up using the output features. This is fixed in the new versions of the files.
Click on the link above for the latest versions of these files.
