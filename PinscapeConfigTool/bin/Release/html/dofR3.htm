Mise à jour DOF R3

Dernière mise à jour
Si vous utilisez DirectOutput Framework, vous devez mettre à niveau vers la dernière version sur le lien ci-dessus.
Vous devez également mettre à jour votre fichier LEDWIZ.DLL si vous l'avez installé. Ceci est utilisé par les anciens logiciels non DOF (tels que LEDBlinky et Future Pinball) pour accéder aux fonctionnalités de LedWiz. Le "stock" LEDWIZ.DLL peut faire que ces programmes plantent si vous avez l'une des fonctionnalités du clavier Pinscape V2 activée.
Vous devez mettre à jour ces fichiers même si vous n'utilisez pas les caractéristiques du contrôleur de sortie. Les anciennes versions peuvent planter suite à la recherche de périphériques s'ils fonctionnent dans une carte Pinscape avec le clavier activé, afin qu'ils puissent planter même s'ils ne finiront pas par utiliser la sortie fonctionnalités. Ceci est corrigé dans les nouvelles versions des fichiers.
Cliquez sur le lien ci-dessus pour les dernières versions de ces fichiers.
