Mode d'apprentissage IR

La carte Pinscape peut apprendre les commandes de la télécommande via son capteur IR. Pour utiliser cette fonction, vous devez configurer le Récepteur infrarouge, comme décrit dans le Guide de construction.
Après avoir ouvert la boîte de dialogue d'apprentissage IR, prenez votre télécommande en la pointant vers le capteur infrarouge Pinscape. Ne pas appuyez encore sur n'importe quel bouton de la télécommande. Cliquez sur le bouton "Apprendre" quand vous êtes prêt. Lorsque vous y êtes invité, appuyez et maintenez le bouton de la télécommande que vous voulez pour que Pinscape apprenne. la fenêtre de dialogue se mettra à jour automatiquement dès que Pinscape apprendra le code, qui ne prend généralement qu'une seconde ou deux.
Si le processus d'apprentissage échoue, vous pouvez cliquer sur "Réessayer" pour lui donner une autre chance. Transmissions IR échoue occasionnellement en raison de problèmes momentanés, tels que des interférences d'une autre source lumineuse. Si Pinscape refuse de reconnaître le code après quelques essais, la télécommande n'est pas compatible avec le matériel ou le logiciel Pinscape. Il existe deux possibilités principales:
  • Si l'affichage du pouls en bas de la boîte de dialogue montre une série de lignes, le capteur détecte correctement le signal IR physique de votre télécommande, mais le logiciel ne reconnaît pas le format des données. Il y a de nombreux formats de données IR, le logiciel Pinscape reconnaît une trentaine d'entre eux, qui couvrent la plupart des grandes sociétés d'électronique, mais il existe de nombreux autres protocoles que Pinscape ne reconnaît pas. Dans ce cas, il y a de l'espoir: le problème est purement une question de logiciel, afin que nous puissions ajouter la prise en charge du nouveau protocole dans une version future, vous pouvez contacter les développeurs Pinscape et nous le faire savoir. Dites-nous la marque et le modèle du téléviseur ou un autre appareil, et incluez le contenu de la fenêtre "Afficher les données brutes".
  • Si l'affichage du pouls en bas de la boîte de dialogue est complètement vide, le capteur ne reçoit aucun signal de la télécommande. Cela peut signifier que votre télécommande utilise une longueur d'onde IR ou une fréquence de modulation que le capteur Pinscape ne peut pas recevoir. Malheureusement, ce type d'incompatibilité est difficile à résoudre, car il faudrait remplacer le capteur par un autre qui peut fonctionner avec votre télécommande. Mais le remplacement du capteur pourrait créer un problème complètement séparé, le logiciel Pinscape est uniquement conçu pour fonctionner avec certains types de capteur. Avant d'abandonner, revérifiez que le capteur lui-même fonctionne, en essayer une télécommande d'un autre fabricant.
