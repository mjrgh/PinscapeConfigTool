Configuration IR

La carte Pinscape peut envoyer et recevoir des commandes IR si vous avez les composants supplémentaires nécessaires attachés. Pour envoyer des commandes, vous avez besoin d'une LED IR. Pour recevoir, vous avez besoin d'un capteur IR (TSOP38438 ou équivalent). Les capacités d'émission et de réception sont indépendantes, vous pouvez donc attachez l'un ou les deux composants. Le guide de construction contient des détails sur ce dont vous avez besoin et comment tout attacher a la KL25Z.

Liste des commandes IR

Vous pouvez programmer l'appareil Pinscape avec une liste de commandes IR pris des télécommandes de votre (vos) téléviseur (s) et autres appareils.
Une "commande" IR est le signal envoyé lorsque vous appuyez sur un bouton sur l'une de vos télécommandes. Chaque bouton a généralement un signal IR unique associé avec. Pinscape peut apprendre ces signaux et les stocker dans la liste des commandes, puis utilisez les commandes stockées pour répétez les mêmes commandes via son émetteur IR, et pour reconnaître les commandes lorsqu'elles arrivent sur son récepteur.
Chaque entrée de la liste des commandes IR comprend les éléments suivants:
  • Code IR: c'est un nombre magique représentant une Commande IR. Dans la plupart des cas, vous obtiendrez ces codes en utilisant la fonction "apprendre".
  • Apprendre (): Cliquez sur ce bouton pour faire apparaître la boîte de dialogue "apprendre", qui permet à Pinscape d'apprendre un code en lisant le signal IR de l'un de vos télécommandes via le récepteur infrarouge Pinscape.
  • Test (): Cliquez sur ce bouton pour tester un code en le transmettant via la LED Pinscape IR. Cela vous permet de confirmer que le périphérique cible reçoit le signal lorsque Pinscape envoie le code.
  • TV ON (): Cliquez dessus pour inclure la commande IR dans la séquence TV ON lorsque le système démarre. Cela vous permet d'allumer votre ou vos téléviseurs avec des commandes IR.
  • Clé (): Cliquez dessus pour sélectionner une touche du clavier ou un bouton du joystick à envoyer sur le PC chaque fois que Pinscape reçoit le code IR. Cela vous permet d'utilisez une télécommande pour envoyer les touches au PC.
  • Description: vous pouvez le remplir avec le nom de la télécommande ou du bouton de commande programmé dans la fente ("Vizio TV ON", par exemple). Ceci est purement facultatif. C'est juste un aide mémoire pour vous aider à garder une trace de ce qui est programmé où, depuis le champ "Code IR" c'est assez cryptique en soi.

Numéros de code IR

Chaque code IR est un nombre magique qui indique au micrologiciel Pinscape comment faire pour lire et envoyer un code de télécommande particulier. C'est "magique" mais c'est vraiment significatif que dans le logiciel Pinscape. le nombre code le type de protocole de signal infrarouge utilisé par la télécommande (ceux-ci varient selon le fabricant, et il y en a beaucoup) et les bits de données contenu dans le signal IR.
La meilleure façon de déterminer un code est d'utiliser la fonction "apprendre", qui permet à Pinscape de déterminer le code d'un bouton particulier de la télécommande en lisant le signal que votre télécommande envoie lorsque vous appuyez sur un bouton. Vous pouvez également taper simplement un code si vous connaissez le numéro, par exemple en obtenant un code d'un autre utilisateur de Pinscape. Mais vous ne pouvez généralement pas utiliser les numéros de code que vous pourriez voir publiés dans tables de codes IR tiers sur Internet, car chaque table utilise son propre système de numérotation qui ne correspond probablement pas au format Pincsape.

Ce que Pinscape peut faire avec IR

Vous pouvez utiliser les capacités IR de plusieurs manières:
  • Allumez votre (vos) téléviseur (s) du pincab au démarrage du système
  • Appuyez sur la touche d'envoi vers le PC lors de la réception d'une commande IR
  • Transmettez des commandes IR lorsque vous appuyez sur un bouton du pincab
Ceux-ci sont tous décrits plus en détail ci-dessous.

TV ON avec IR

L'émetteur IR peut être utilisé pour allumer les téléviseurs de votre pincab lorsque le système démarre. Il s'agit d'une alternative moins invasive à la fonction TV ON câblée (où vous devez ouvrir le boîtier du téléviseur et soudez les fils à l'interrupteur «marche» de votre téléviseur). Avec IR, vous n'avez pas à faire d'une intervention chirurgicale à la télévision.
Si vous n'avez pas encore configuré les paramètres de base TV ON, vous devez d'abord les configurer. Accédez à la section TV ON dans les paramètres, activez-le, réglez la minuterie de retard et configurez les broches d'entrée. Si vous n'utilisez pas les cartes d'extension Pinscape, vous devrez construire des circuits externes, ce qui est expliqué dans le Guide de construction. Les parties «Entrée d'état de l'alimentation» et «Sortie de verrouillage d'état» sont requis. La "Sortie relais" n'est pas requise si vous utilisez seulement l'IR (mais il est également possible d'utiliser à la fois l'IR et le relais; ils sont heureux de travailler ensemble ou séparément).
Ensuite, configurez la LED de l'émetteur Pinscape IR et positionnez-la de manière à ce que votre téléviseur puisse voir les commandes qu'il émet. Voir le guide de construction pour obtenir des instructions sur le câblage de la LED IR. Positionnez la LED de sorte qu'elle soit près de la fenêtre du capteur de la télécommande et pointe vers la télé.
Il ne reste plus qu'à programmer la commande "ON" du téléviseur dans un emplacement de la liste des codes IR. Sortez la télécommande de votre téléviseur et programmez le bouton ON dans une fente comme décrit ci-dessus. Cliquez sur l'icône TV ON () pour inclure la commande dans la séquence de mise sous tension. Pinscape transmettera cette commande via son émetteur IR lorsque l'alimentation se mettra en marche après la fin de la minuterie.

Envoyer une touche lors de la réception d'une commande IR

Vous pouvez programmer l'unité Pinscape pour envoyer une pression de touche au PC pour recevoir une commande IR.
Par exemple, supposons que vous souhaitiez pouvoir régler le volume audio sur votre PC, mais vous n'avez pas de boutons de rechange sur votre armoire. Et supposons que vous ayez une vieille télécommande qui traîne pour un appereil que vous n'utilisez plus, et il a une sorte de touches Haut et Bas. Vous pouvez programmer Pinscape pour qu'il envoie une touche de clavier "Augmenter le volume" sur le PC lorsque vous appuyez sur le bouton "Haut" de la télécommande et une autre touche du clavier "Baisser le volume" lorsque vous appuyez sur "Bas" de la télécommande.
Pour utiliser cette fonction, vous devrez configurer le capteur infrarouge Pinscape, et le positionner de manière à ce qu'il puisse voir la lumière infrarouge de votre télécommande. Cela signifie, positionner le capteur à l'extérieur du pincab, ou à travers un trou dans le pincab. Vous ne pouvez pas le placer entièrement à l'intérieur de pincab, car La lumière infrarouge de la télécommande ne pénètre pas dans les parois de l'armoire.
Accédez maintenant à la liste des commandes IR dans les paramètres. Programme dans le codes des boutons de la télécommande, comme décrit ci-dessus, pour les boutons que vous souhaitez transformer en frappes. Pour chacun d'eux, cliquez sur le Icône "Clé" () pour sélectionner la touche à envoyer. Une fois que tout est configuré, Pinscape "appuiera" sur la touche programmée chaque fois qu'elle recevra le Commande IR.

Envoyer une commande IR lorsque vous appuyez sur un bouton du cabinet

Vous pouvez programmer Pinscape pour transmettre une commande IR à chaque fois que vous appuyez sur un bouton du pincab.
Par exemple, supposons que vous souhaitiez pouvoir régler le volume de votre téléviseur via les boutons du pincab. Pour ce faire, vous pouvez configurer des boutons sur votre pincab pour envoyer des commandes d'augmentation et de diminution du volume au téléviseur via IR.
Pour faire ce travail, vous devez configurer la LED de l'émetteur Pinscape IR, comme expliqué dans le Guide de construction et positionnez le voyant de sorte que le Le capteur de la télécommande du téléviseur puisse le voir. C'est le même arrangement nécessaire pour la fonction TV ON via IR, donc vous avez probablement déjà dû mettre cela en place.
Il vous suffit maintenant d'assigner une commande IR à chaque bouton que vous souhaitez utiliser pour envoyer des signaux IR au téléviseur. La première étape consiste à programmer le codes de commande IR que vous souhaitez envoyer. Programmer les commandes souhaitées dans les emplacements de la liste de commandes IR comme décrit ci-dessus.
Une fois que vous avez programmé les commandes IR, accédez à la section "Boutons" dans les paramètres. Recherchez la ligne du bouton que vous souhaitez utiliser pour envoyer une commande IR, et cliquez sur l'icône "Commande IR" dans la ligne (). Sélectionner la commande IR souhaitée dans le menu.
