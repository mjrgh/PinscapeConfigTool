Testeur de port de sortie

Cette fenêtre vous permet de tester le câblage du port de sortie physique sur votre appareil Pinscape.

La liste montre chaque port de sortie et son emplacement de broche sur la KL25Z, la puce externe ou les cartes d'extension.

Si vous avez besoin d'un rappel de l'emplacement de la broche pour un port de sortie, déplacez la souris sur la ligne du port dans la liste à gauche. Cela mettra en évidence l'emplacement de la broche sur le schéma du KL25Z ou de l'extension à droite.

Les ports peuvent être compatibles PWM ou numériques. Les sorties PWM peuvent être réglées sur une plage de niveaux de luminosité / intensité, tandis que les sorties numériques peuvent uniquement être activées et désactivées. Le type de chaque port dépend du matériel de sortie physique et de votre sélection sur la page "Paramètres". Toutes les broches GPIO de la KL25Z peuvent être utilisé comme sorties numériques, alors que seul un sous-ensemble de broches GPIO sont compatibles PWM, car le matériel de la KL25Z ne fournit le câblage interne nécessaire pour PWM sur certaines broches. La page "Paramètres" vous permet d'attribuer le mode (PWM ou numérique) pour chaque broche, et ne vous permet de sélectionner PWM que pour les broches qui ont le câblage interne nécessaire. Les ports TLC5940 sont toujours des ports PWM et 74HC595 sont toujours numériques.

Pour un port PWM, la liste de sortie affiche un curseur de contrôle qui vous permet de définir un niveau de luminosité pour le port de 0 à 255. 0 est OFF et 255 est une luminosité de 100%. Il y a aussi un bouton ON / OFF à côté de chaque curseur que vous pouvez utiliser pour activer et désactiver le port sans changer le réglage de la luminosité.

Pour un port numérique, la liste comprend uniquement un bouton ON / OFF. Cliquez sur le bouton pour basculer l'état de sortie.

Mode nuit

Le bouton Mode nuit vous permet d'activer et de désactiver le mode nuit. Cela vous permet de vérifier que les bonnes sorties sont marquées comme "bruyant" dans la configuration du port et ce mode nuit les désactive correctement lorsqu'ils sont engagés. Lorsque le mode nuit est activé, tous les ports marqués «bruyant» dans la configuration sont désactivés, donc ces sorties physiques resteront éteintes même si vous les allumez ou augmenter leur luminosité dans la liste. Remarqué que le logiciel marche / arrêt et les paramètres de luminosité pour les ports "bruyants" ne sont pas affectés, les ports sont simplement coupé au niveau des broches de sortie physiques. Donc vous pouvez encore régler l'activation / la désactivation du logiciel et les paramètres de luminosité pour les ports bruyants lorsque le mode nuit est activé, et ces paramètres seront appliqués aux ports physiques dès que vous désactiver le mode nuit.

Dépannage de Visual Pinball, DOF et autres émulateurs

La fenêtre Testeur de sortie peut être utile si vous rencontrez des problèmes pour obtenir vos périphériques de sortie pour fonctionner avec votre émulateur de flipper.

Le testeur de sortie communique directement avec la KL25Z via USB. Cela ne dépend d'aucun autre logiciel partagé dont vous pourriez avoir besoin pour d'autres programmes, tels que DOF, LEDWIZ.DLL, etc. Cela permet d'isoler plus facilement les source de tout problème:

  • Si les sorties fonctionnent correctement dans la fenêtre du testeur de sortie, il confirme que votre câblage physique fonctionne correctement, que la KL25Z est correctement configuré et que la connexion USB vers la KL25Z est fonctionnelle. Cela exclut les problèmes avec votre configuration matérielle donc tout problème avec d'autres logiciels doit se trouver dans les autres logiciels. Si vous utilisez Visual Pinball, l'endroit où commencer à chercher est généralement le DOF (DirectOutput Framework). DOF a sa propre configuration que vous avez à configurer pour correspondre à la configuration de votre port Pinscape.

    Pour obtenir de l'aide sur les problèmes de DOF, consultez le Dépannage DOF dans le guide de construction Pinscape.

  • Si les sorties ne fonctionnent pas dans le testeur de sortie, il y a probablement quelque chose qui ne va pas avec le câblage physique ou avec la configuration du micrologiciel Pinscape. La première étape consiste à vérifiez que les sorties sont correctement configurées dans la page "Paramètres". Assurez-vous que le câblage physique correspond aux affectations des broches, et qu'il n'y a pas d'erreur ou d'icônes d'avertissement affichées à côté des attributions de broches n'importe où sur la page "Paramètres". L'outil effectue de nombreuses vérifications d'erreurs pour vous aider, assurez-vous que la configuration est correcte. Il vérifie que vous n'avez pas assigné la même broche à plusieurs fonctions et que chaque broche est capable de remplir sa fonction assignée. Si tout est "OK" dans les "paramètres", le prochain endroit à regarder est le câblage physique. Commencez par vérifier que c'est le câblage que vous vouliez: que les bonnes choses soient connectées aux bonnes broches, blocs d'alimentation, etc. Si tout correspond à vos plans, la prochaine étape est de sortir un voltmètre et de vérifier la continuité à travers chaque circuit, pour s'assurer qu'il 'y a pas de fils coupés, de fils cassés ou de mauvaise soudure de connexions.
